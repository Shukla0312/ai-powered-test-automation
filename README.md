# AI-Powered Test Automation Framework

## 🚀 Overview
This project demonstrates how AI (LLMs) can enhance traditional test automation by enabling intelligent validation, test generation, and flaky test detection.

## ✨ Key Features
- AI-based API response validation
- LLM-driven test case generation
- Flaky test detection logic
- Modular and scalable architecture

## 🧠 Why This Matters
Traditional automation validates exact values.
This framework uses AI to validate correctness semantically, reducing brittle tests and improving coverage.

## 🛠️ Tech Stack
- Node.js

- OpenAI API
