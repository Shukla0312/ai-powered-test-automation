TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════

Problem: "OPENAI_API_KEY environment variable is required"
Solution: Add OPENAI_API_KEY=sk-... to your .env file

Problem: "Module not found"
Solution: Run: npm install

Problem: "Tests run but fail validations"
Solution: Lower minValidationScore in SemanticValidator constructor
          (it's set to 75 by default, try 60)

Problem: "API requests fail/timeout"
Solution: Check API_BASE_URL is correct and accessible
          Check internet connection
          Check firewall isn't blocking API.OpenAI.com
